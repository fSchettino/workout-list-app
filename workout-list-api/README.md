# Workout List API

Node JS API for Workout List Client.<br>

## Key features:

TDD - Test Driven Development.

## Built With:

Node Js<br>
Express<br>
SQLite<br>
Mocha<br>
Chai<br>

## Commands:

- Run API > "npm start".
- Run tests > "npm run test".

Open http://localhost:8000/ to check server running.
